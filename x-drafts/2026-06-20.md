## パターン1：気づき・学び型

Claude Code 2.1.183で破壊的なgitコマンド（reset --hard, clean -fd）が明示的指示なしに自動ブロックされるように。副業でコードを触る時って集中力が本業より落ちがちで、うっかりミスが怖い。AIが安全弁になってくれるの、地味に助かる。

## パターン2：実践・体験談型

Claude Code ArtifactsはAIがセッション中に共有可能なWebページを自動生成・更新してくれる機能。副業クライアントへの進捗報告って意外と時間かかるんですよね。「今ここまで動いてます」って見せられるページを自動で作れたら、報告コストが激減しそうだと思う。

#### Desktop Redesign
- **マルチセッションサイドバー**: 並列タスクを同時進行。各セッションは `<project-root>/.claude/worktrees/` に独立した Git ワークツリーを作成しコンフリクト回避

- 要件: Claude Desktop v1.2581.0 以降、Pro / Max / Team / Enterprise 対象
- ソース: https://www.buildfastwithai.com/blogs/claude-code-desktop-redesign-2026

#### Claude Code Routines（クラウド自動化）
- Anthropic インフラ上で動作するスケジュール実行機能。**PC がオフラインでも実行可能**
- トリガー 3 種: スケジュール（時間・日・週）、API エンドポイント、GitHub イベント（PR / push / issue / release）
- プラン別実行回数: Pro=**5 回/日**、Max=**15 回/日**、Team/Enterprise=**25 回/日**
- 活用例: 夜間バックログトリアージ、PR 自動レビュー、ドキュメントドリフト検出
- ※ GitHub Actions（確定的 CI/CD）とは補完関係。推論が必要な解釈的タスクを担う
- ソース: https://9to5mac.com/2026/04/14/anthropic-adds-repeatable-routines-feature-to-claude-code-heres-how-it-works/

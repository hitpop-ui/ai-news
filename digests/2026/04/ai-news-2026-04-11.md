# AI News Digest — 2026-04-11

## Claude Code

### Claude Code 2.1.101（2026-04-10）— チームオンボーディング・セキュリティ強化
- **`/team-onboarding` コマンド追加**: ローカルのClaude Code使用履歴からチームメイト向けオンボーディングガイドを自動生成
- **OS CA証明書ストアをデフォルト信頼**: エンタープライズTLSプロキシが追加設定なしで動作。`CLAUDE_CODE_CERT_STORE=bundled` でバンドルCAのみに切り替え可能
- **`/ultraplan` 自動環境作成**: リモートセッション機能がデフォルトクラウド環境を自動作成（事前Web設定不要）
- **セキュリティ修正**: POSIX `which` フォールバックのコマンドインジェクション脆弱性を修正
- **バグ修正**: 長時間セッションのメモリリーク・`--resume`/`--continue` での大規模セッションコンテキスト消失・5分ハードコードタイムアウト問題を修正
- ソース: https://code.claude.com/docs/en/changelog

### Claude Code 2.1.98（2026-04-09）— Vertex AI・Monitor・サンドボックス
- **Google Vertex AI対話型セットアップウィザード**: ログイン画面の "3rd-party platform" 選択時にGCP認証を案内
- **Monitor ツール**: バックグラウンドスクリプトからのイベントストリーミングが可能に
- **Linuxサブプロセスサンドボックス**: PIDネームスペース分離 + `CLAUDE_CODE_SUBPROCESS_ENV_SCRUB`
- **セキュリティ修正（重要）**: バックスラッシュ転送によるBashツール権限バイパス・複合コマンドによる権限チェック回避・`/dev/tcp//dev/udp/` へのリダイレクトが確認なしで通過していた問題を修正
- **429レート制限**: 指数バックオフが正しく適用されるよう修正（旧: 約13秒で全リトライを消費）
- ソース: https://code.claude.com/docs/en/changelog

# AI News Digest - 2026-04-24

## Claude Code

### v2.1.118 リリース（2026-04-23）
- **Vim ビジュアルモード追加**: `v`（ビジュアル）・`V`（行選択）、選択・オペレータ・ビジュアルフィードバック対応
- `/cost`・`/stats` → **`/usage` に統合**（両コマンドはショートカットとして継続）
- **名前付きカスタムテーマ**: `/theme` から選択 または `~/.claude/themes/` に JSON を置いて定義
- フック内で MCP ツール直接呼び出し可能（`type: "mcp_tool"`）
- `/color` でセッションアクセント色を claude.ai/code に同期
- `DISABLE_UPDATES` 環境変数で全アップデート経路をブロック可能
- WSL on Windows で Windows 側の管理設定を継承（`wslInheritsWindowsSettings`）
- OAuth トークンリフレッシュ時の競合状態を複数修正、macOS キーチェーン・Linux/Windows 認証情報クラッシュ修正
- URL: https://code.claude.com/docs/en/changelog

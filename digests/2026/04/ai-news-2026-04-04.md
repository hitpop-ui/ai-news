# AI News Digest — 2026-04-04

## Claude Code

### Claude Code v2.1.91（4月2日リリース）★最重要
- **MCP tool result persistence override**: MCPツール結果に `_meta["anthropic/maxResultSizeChars"]` アノテーションで最大500Kまでの大きな結果（DBスキーマ等）を截断なしで通過可能に
- **`disableSkillShellExecution`**: スキル・カスタムスラッシュコマンド・プラグインコマンドのインラインシェル実行を無効化する設定を追加
- **ディープリンクのマルチライン対応**: `claude-cli://open?q=` で最大5,000文字のエンコード改行（`%0A`）をサポート
- **プラグイン実行可能ファイル**: `bin/` ディレクトリ内の実行可能ファイルを Bash ツールからベアコマンドとして呼び出し可能に
- **バグ修正**: `--resume` 時のトランスクリプトチェーンブレイク修正、リモートセッションのプランファイル追跡失敗修正
- URL: https://code.claude.com/docs/en/changelog

### Claude Code v2.1.90（4月1日リリース）
- **`/powerup` コマンド追加**: Claude Code 機能を対話的に学べるアニメーション付きレッスン機能
- レート制限ダイアログが無限ループする不具合を修正
- `--resume` 時のプロンプトキャッシュミス問題を修正
- URL: https://code.claude.com/docs/en/changelog

## Cursor

### Cursor 3.0 リリース（4月2日）★最重要
- **Agents Window**: ローカル・worktrees・クラウド・リモート SSH など異なる環境で複数エージェントを並列実行
- **Design Mode**: ブラウザ上でUI要素に直接アノテーションを付与し、精密なフィードバックで高速イテレーション
- **Agent Tabs**: 複数チャットをサイドバイサイドまたはグリッドレイアウトで表示
- エンタープライズ向けにコードアトリビューション管理・シークレット管理のコントロールを強化
- URL: https://cursor.com/changelog

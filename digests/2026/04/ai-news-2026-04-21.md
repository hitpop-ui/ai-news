### xAI: Grok Speech-to-Text / Text-to-Speech API リリース（2026-04-17）

xAI が音声 API を公開。Grok Voice・Tesla 車載システム・Starlink サポートと同技術を開発者向けに開放。

- **STT**: リアルタイム＆バッチ処理、25+ 言語対応、話者識別あり、$0.10〜0.20/時間
- **TTS**: 感情タグ（`[laugh]`・`[whisper]`）付き表現力豊かな音声生成、$4.20/100 万文字
- 低遅延を特徴とし、エージェント・音声インターフェース向け
- URL: https://releasebot.io/updates/xai

### Cloudflare Agents Week 総括（2026-04-13〜17）

Cloudflare が 1 週間でエージェントスタック全レイヤーを刷新。前回チェック時に未収録のため補足。

- **Dynamic Workers**: AI 生成コードを安全・高速に実行するミリ秒起動の isolate ランタイム
- **Sandboxes GA**: エージェント用の永続 Linux 環境が GA
- **Cloudflare Mesh**: エージェント間のゼロトラスト・プライベートネットワーク
- **統合 AI 推論レイヤー**: 14 プロバイダー・70+ モデルを 1 つの API・1 クレジットで切り替え可能
- **Agent Memory**: 永続メモリのマネージドサービス（記憶・忘却・学習）
- **Artifacts**: エージェント向け Git 互換バージョン管理ストレージ
- **Unweight**: 推論時の無損失圧縮で最大 22% モデルフットプリント削減→高速・低コスト化
- **Browser Run**: エージェントによる Web コンテンツ操作
- **次世代 Agents SDK**: バッテリー込みのエージェントプラットフォームをプレビュー公開
- URL: https://blog.cloudflare.com/agents-week-in-review/

# AI News Digest — 2026-04-02

## Claude Code

### Claude Code v2.1.89（4月1日リリース）★最重要
- **デファード権限決定**: `PreToolUse` フックに `"defer"` 決定を追加。ヘッドレスセッションがツール呼び出しで一時停止し、`-p --resume` で再評価可能に
- **破壊的変更**: 思考サマリーがインタラクティブセッションでデフォルト無効化。復元には `showThinkingSummaries: true` を settings.json に追加
- **`PermissionDenied` フック**: 自動モード分類器の拒否後に発火。`{retry: true}` 返却でモデルに再試行を許可
- **名前付きサブエージェント**: `@` メンションの候補に表示
- **`MCP_CONNECTION_NONBLOCKING=true`**: `-p` モードで MCP 接続待ちをスキップ
- **バグ修正多数**: autocompact ループ、メモリリーク、Windows CRLF、LSP ゾンビ状態等
- URL: https://code.claude.com/docs/en/changelog

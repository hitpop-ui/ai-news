## Claude Code

### Claude Code 2.1.101（4/10）
- `/team-onboarding` コマンド追加：ローカルのClaude Code使用状況からチームメンバー向けのオンボーディングガイドを自動生成
- OSのCAストアをデフォルトで信頼するよう変更（企業TLSプロキシの追加設定が不要に）。バンドルCAのみ使うには `CLAUDE_CODE_CERT_STORE=bundled`
- `/ultraplan` 等のリモートセッション機能がデフォルトクラウド環境を自動生成（Webで事前設定が不要に）
- **セキュリティ修正**: LSPバイナリ検出のPOSIX `which` フォールバックにあったコマンドインジェクション脆弱性を修正
- 長セッションでのメモリリーク修正、`--resume`/`--continue` の大規模セッションでのコンテキスト喪失修正
- `permissions.deny` ルールがPreToolUseフックに上書きされていたバグ修正
- サブエージェントが動的注入MCP toolsを継承できない問題を修正
- https://code.claude.com/docs/en/changelog

### Claude Code 2.1.98（4/9）
- Google Vertex AI の認証・設定インタラクティブウィザード追加
- `CLAUDE_CODE_PERFORCE_MODE` 環境変数：Perforce環境での読み取り専用ファイルを `p4 edit` ヒント付きで扱える
- Monitor tool：バックグラウンドスクリプトのイベントをストリーミング取得
- Linux上でPIDネームスペース隔離によるサブプロセスサンドボックス化
- **セキュリティ修正**: バックスラッシュエスケープフラグによるBashツール権限バイパスを修正、複合Bashコマンドが強制権限プロンプトを回避していた問題を修正
- https://code.claude.com/docs/en/changelog

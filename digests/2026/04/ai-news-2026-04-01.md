# AI News Digest — 2026-04-01

## 最重要

### Claude Code v2.1.88 リリース（2026-03-30）
- `CLAUDE_CODE_NO_FLICKER=1` 環境変数追加：altスクリーン描画のちらつきを抑制
- `PermissionDenied` フックを新設：autoモードのClassifierによる拒否後に発火
- サブエージェントへの命名機能追加：`@` メンションのタイプアヘッドに表示
- バグ修正：長時間セッションでのプロンプトキャッシュミス、ネストした`CLAUDE.md`の再注入、Windows上でのCRLF二重化
- ソース: https://code.claude.com/docs/en/changelog

### Claude Code ソースコード漏洩事件（2026-03-31）
- npmパッケージに誤ってソースマップファイルが同梱され、TypeScript全ソース約512,000行・1,900ファイルが露出
- セキュリティ研究者のChaofan Shou氏がX上で発見・報告
- Anthropicは即座にnpmアップデートを配布しソースマップを削除、旧バージョンをnpmレジストリから取り下げ
- 漏洩コードに未公開機能が確認：**KAIROS**（常時起動型Claude）、**ULTRAPLAN**（30分リモートプランニング）、**BUDDY**（デジタルペット、4/1〜7リリース窓・5月本格展開予定）等
- ソース: https://dev.to/evan-dong/claude-codes-entire-source-code-just-leaked-512000-lines-exposed-3139

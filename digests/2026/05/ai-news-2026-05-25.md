## DeepSeek Code Harness チーム正式結成（2026-05-20確認）

DeepSeek上級研究員 Chen Deli がSNSで「Claude Codeに直接対抗する『DeepSeek Code Harness』の社内チームを正式結成」と確認。元Jane Streetエンジニア Cui Tianyi を3月に採用済み。2026-05-20にPM・R&Dエンジニアの求人を北京拠点向けに公開。

- **コンセプト**: 「Model + Harness = Agent」—DeepSeek V4の上にエージェント向けワークフロー管理層を構築
- **目標**: 自律的なプランニング・コーディング・テスト・デバッグ。個別提案ではなくプロジェクト全体の完結
- **現状**: チーム構築フェーズ。製品ローンチまで6〜12ヶ月以上必要と報道
- **背景**: DeepSeek V4 Pro（$1.74/1M入力）の低価格とClaude Codeの市場シェアを踏まえた競合ポジショニング

## Kimi K2.5 サポート終了（本日 2026-05-25）

Moonshot AIのKimi K2.5のAPIサポートが本日をもって終了。全ユーザーはK2.6への移行が必要。

- **K2.6の主な仕様**: 1T MoE / 32Bアクティブパラメータ・256Kコンテキスト・最大1,000エージェントスワーム対応
- **価格**: $0.60/$2.50 per 1M tokens（input/output）
- K2.5を使用しているワークフロー・スクリプトの移行対応が本日必要

> https://www.kimi.com/ai-models/kimi-k2-6

# AI News Digest 2026-05-17

## コーディングエージェント

### xAI が Grok Build ベータを公開（2026-05-14〜15）
Claude Code・OpenAI Codex CLI に続く3つ目の主要コーディングエージェントCLIが登場。Grok 4.3 beta搭載、最大8並列エージェント、2Mトークンコンテキスト対応。アプリ構築・ワークフロー自動化まで自律的に実行可能。
- 提供対象: SuperGrok Heavy（通常$300/月、初月6ヶ月は$99/月の割引あり）
- 出典: https://x.ai/news/grok-build-cli / https://devops.com/xai-enters-the-coding-agent-race-with-grok-build/

### Claude Code v2.1.143（2026-05-15）
前回確認の v2.1.142 から更新。プラグイン依存関係管理を強化（`claude plugin enable` で推移的依存を自動有効化、`disable` 時は依存するプラグインがあれば拒否）。`/plugin` マーケットプレイスに投影コスト表示を追加。`worktree.bgIsolation: "none"` 設定でバックグラウンドセッションが EnterWorktree なしに編集可能に。PowerShell に `-ExecutionPolicy Bypass` をデフォルトで渡すよう変更。
- 出典: https://code.claude.com/docs/en/changelog

## OpenAI / ChatGPT

### ChatGPT Personal Finance（2026-05-15）
米国の Pro ユーザー向けに個人財務機能をプレビュー公開。Plaid 経由で12,000以上の金融機関（Schwab・Fidelity・Chase・Robinhood・AmEx等）に接続し、支出ダッシュボード・サブスクリプション管理・将来支払い予測を提供。「財務メモリ」で文脈を保持した継続的な財務Q&A が可能。4月に買収した個人財務スタートアップ Hiro のチームが開発。Web・iOS で利用可能（Plus 展開は後日）。
- 出典: https://openai.com/index/personal-finance-chatgpt/

## Devin

### Devin May 1 アップデート（2026-05-01）
前回チェック（4/24）以降の新機能まとめ。

- **段階的レビュー権限**：企業管理者が組織メンバーへ PR レビューアクセスレベルを段階的に割り当て可能に
- **ネットワークポリシー**：セッション中に Devin がアクセスできるホスト・IP 範囲を制限できるポリシー機能を追加
- **スキルスラッシュコマンド**：`@mention` 構文に加え、`/name` でスキルを直接呼び出せるように
- **大規模テキスト自動添付**：大きなテキストをプロンプト欄へ貼り付けるとテキストボックスではなくファイルとして自動添付
- **Jira プロジェクトマッピング再設計**：固定ヘッダー＋スクロール可能コンテンツで UI 改善
- **オート修正が CI チェックを含む**：失敗した CI チェック名もレビュー結果に含まれ、修正精度が向上
- **v3 API に `origin` フィールド追加**：セッション起源（webapp / Slack / API / CLI）を返す
- **宣言的設定スナップショット復帰**：`ManageOrgSnapshotsPermission` でクラシック設定への巻き戻しが可能に

- URL: https://docs.devin.ai/release-notes/overview

### Warp：AIエージェント専用ターミナル開発環境（2026-05-02）

## Claude Code

### Claude Code 2.1.128 リリース（2026-05-05）
前回確認バージョン 2.1.126 から 2 バージョン進んだ。主な変更点：

- `/color`（引数なし）でランダムセッションカラーを自動選択
- `/mcp` コマンドがサーバーごとのツール数を表示、ツール 0 のサーバーをフラグで識別
- `.zip` 形式プラグインアーカイブに対応
- コンソール認証でのチャネル機能サポート追加
- 10MB 超の入力パイプ時クラッシュ・画像アップロード時ハングなど主要バグ修正
- `OTEL_*` 環境変数がサブプロセス（Bash/MCP等）に継承されなくなりテレメトリ漏洩を防止
- `workspace` が MCP サーバー名として予約語化

URL: https://code.claude.com/docs/en/changelog

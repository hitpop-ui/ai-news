## Devin（大型アップデート 2026-05-08）

10 項目が一挙公開。副業・開発チーム双方に影響大：

- **Devin Review API**：`POST /v3/organizations/{org_id}/pr-reviews` で CI/CD パイプラインからプログラマティックにコードレビューをトリガー可能。人手を介さないレビュー自動化が現実に
- **Mermaid 図レンダリング**：セッション内コードブロックがズーム・パン付き SVG として表示
- **セッションアーカイブ時の PR 自動クローズ**：リンクされた GitHub PR を連動クローズ
- **PR 別自動レビュー切り替え**：PR 単位で Devin Review の有効/無効を設定可能
- **「Ask Devin」ボタン**：PR ディスカッションのスレッドコメント内から直接 Devin に質問
- **MCP シークレットスコープ**：個人・組織スコープを選択可能になり権限管理が細分化
- **Service User パーミス管理**：エンタープライズ管理者が ManageAccountServiceUsers 権限を割り当て可能
- URL: https://docs.devin.ai/release-notes/overview

## OpenAI

- **Codex v0.130.0**（2026-05-09）：`codex remote-control` コマンドでヘッドレスのリモート制御可能アプリサーバーを起動可能に。プラグイン共有機能（リンクメタデータ・発見可能性制御）、AWS Bedrock 認証（IAM 認証情報対応）、大規模スレッドのページネーション、マルチ環境セッション時の画像解像度向上
- **Codex Chrome Extension**（2026-05-07）：複数タブ並行動作、サイトアクセス制御をユーザーが保持するブラウザ統合ツール。ターミナル不要でブラウザから直接 Codex を利用可能
- **ChatGPT Enterprise: Workspace Agents**（2026-05-07）：テンプレートから構築・プレビュー・共有・スケジュール実行が可能。ChatGPT × Slack 統合で反復的ワークフローを自動化（EKM 対応企業向け）
- URL: https://releasebot.io/updates/openai

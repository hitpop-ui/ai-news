# AI News Digest — 2026-05-02

## Claude Code

### Claude Code v2.1.126 リリース（2026-05-01）
大型リリース。主な新機能・修正点：

- **`claude project purge [path]`** コマンド追加：トランスクリプト・タスク・ファイル履歴・設定を一括削除（`--dry-run` / `-y` / `-i` / `--all` 対応）
- **Windows PowerShell 7 対応**：Microsoft Store / MSI / .NET global tool インストール版を自動検出。有効時は Bash の代わりに PowerShell がプライマリシェルに
- **`/model` ピッカー改善**：`ANTHROPIC_BASE_URL` が互換ゲートウェイを指す場合、ゲートウェイの `/v1/models` エンドポイントからモデルを取得
- **OAuth 認証改善**：ブラウザコールバックが localhost に届かない環境でターミナルへのコード貼り付けに対応
- **iTerm2 クリップボード自動設定**：`/terminal-setup` 実行で「アプリのクリップボードアクセス」を自動有効化（tmux 含む）
- **MCP サーバー自動リトライ**：起動時の一過性エラーで最大 3 回自動リトライ（従来は切断のまま）
- **セキュリティ修正**：`allowManagedDomainsOnly` / `allowManagedReadPathsOnly` が上位設定でサンドボックスブロックを欠く場合に無視される問題を修正
- **主なバグ修正**：画像キャッシュリーク（マルチ GB RSS 成長）・`/usage` メモリリーク（〜2 GB）・ファイル監視メモリリーク・日本語/韓国語/中国語テキスト破損（Windows）・`Ctrl+L` 動作修正

- URL: https://code.claude.com/docs/en/changelog

## Cursor

### Cursor Team Marketplace 更新（2026-05-01）
管理者がリポジトリ接続なしでチームマーケットプレイスを構築可能に。

- **プラグイン配布モード 3 種**：Default Off（任意導入）/ Default On（任意削除可）/ Required（削除不可の強制インストール）
- プラグインに MCP サーバー・スキル・サブエージェント・ルール・フックをバンドル可能
- Cursor ダッシュボードの plugins セクションから設定

- URL: https://cursor.com/changelog

## Claude Code

### Claude Code 2.1.144（2026-05-19）
バックグラウンドセッションの `/resume` 対応が最大の目玉。`claude --bg` やエージェントビューで開始したセッションを再接続可能に。バックグラウンドサブエージェント完了通知に経過時間表示（例："Agent completed · 3h 2m 5s"）を追加。「extra usage」を「usage credits」に改称（`/extra-usage` は後方互換維持）。MCPサーバーの paginated `tools/list` 対応（先頭ページしか返さない問題を修正）、macOS フルディスクアクセス保護下でのバックグラウンドセッションクラッシュ修正、Windows スクロール・Ghost characters・キーボード入力の複数改善。
- ソース: https://code.claude.com/docs/en/changelog

## Cursor

### Cursor × Jira 統合（2026-05-19）
JiraのワークアイテムをCursorのクラウドエージェントに直接割り当て可能。チケットのコメント欄に「@Cursor」と記述するだけでエージェントが起動し、タイトル・説明・コメントを読み込んでバグ修正・機能追加・テスト更新を実行。Composer 2.5（5/18）から2日での追加リリース。
- ソース: https://cursor.com/changelog

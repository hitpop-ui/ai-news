# AI News Digest — 2026-05-12

## Claude Code

### Claude Code v2.1.139（2026-05-11）
- **エージェントビュー（Research Preview）**: `claude agents` コマンドで全セッション（実行中・ブロック中・完了）を一覧表示
- **`/goal` コマンド**: 完了条件を設定すると Claude が複数ターン自動実行し、経過時間・ターン数・トークン使用量をオーバーレイ表示。インタラクティブ・`-p` モード・Remote Control に対応
- **その他**: `/scroll-speed`（スクロール速度調整）、`claude plugin details <name>`（トークンコスト表示）、`?` キーによるショートカット表示、`{`/`}` でプロンプト間移動
- **バグ修正**: 期限切れ認証によるデッドロック解消・MCP サーバーメモリ増殖修正・ハイパーリンクのダークテーマ対応
- URL: https://code.claude.com/docs/en/changelog

### Code w/ Claude 2026 発表内容（まとめ）
- **Managed Agents 3大機能**: Multiagent Orchestration（エージェント群によるタスク分解）・Outcomes（達成基準を設定し反復改善）・Dreaming（過去セッションを振り返り自律改善）
- **プラグイン強化**: `/skills`・`/plugin` でリアルタイムフィルタ。プラグインがスラッシュコマンド・フック・サブエージェント・スキルを束ねて登録可能
- **コスト削減**: `ENABLE_PROMPT_CACHING_1H`（1時間 TTL キャッシュ）・`FORCE_PROMPT_CACHING_5M` で長時間セッションのコストを最大 3〜4 倍削減可能
- **Git Worktree 統合**: ワークツリーを AI エージェントのファーストクラス機能として組み込み
- URL: https://code.claude.com/docs/en/whats-new

## Cursor

### Microsoft Teams 連携（2026-05-11）
- Teams チャンネルで `@Cursor` メンションするとクラウドエージェントにタスクを委任。コンテキストに応じてリポジトリ・モデルを自動選択し、PR を作成してレビューに回す
- URL: https://cursor.com/changelog

### Bugbot Effort Levels（2026-05-11）
- バグ検出強度を Default・High・Custom の 3 段階に設定可能。Default で平均 0.7 件/実行、High で 0.95 件/実行。カスタマイズは usage-based billing が必要
- URL: https://cursor.com/changelog

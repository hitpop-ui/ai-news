## Claude Code

### Claude Code v2.1.145〜v2.1.152（2026-05-19〜2026-05-27）
8日間で8バージョンという急ピッチリリース。主要変更をバージョン別にまとめる。

**v2.1.152（2026-05-27）— 最新**
- `/code-review --fix`：レビュー指摘をワーキングツリーへ直接適用（`/simplify`も同コマンド呼び出しに統合）
- Skills/スラッシュコマンドがフロントマターで `disallowed-tools` を指定可能に（アクティブ時に特定ツールを無効化）
- `/reload-skills`：セッション再起動なしでスキルディレクトリを再スキャン
- `SessionStart` フックが `reloadSkills: true` + `hookSpecificOutput.sessionTitle` でセッションタイトルを動的設定可能に
- `MessageDisplay` フックイベント追加（アシスタントメッセージを表示時に変換・非表示化）
- `pluginSuggestionMarketplaces` 企業管理設定追加
- Vimモード：NORMALモードで `/` がリバース履歴検索を起動
- Thinkingサマリー：最低3秒表示保証・Markdown形式・10行上限

**v2.1.149（2026-05-22）**
- `/usage` にカテゴリ別コスト内訳（skills / subagents / plugins / MCP サーバーごと）
- `/diff` 詳細ビューがキーボード操作でスクロール可能に
- GFM タスクリストチェックボックスをMarkdown出力でレンダリング
- `allowAllClaudeAiMcps` 企業管理設定追加

**v2.1.147（2026-05-21）**

- ピン留めセッションがアイドル時も存続（`claude agents` の Ctrl+T）
- `/simplify`→`/code-review` リネーム（effort level 指定可・GitHub PRインラインコメント投稿対応）

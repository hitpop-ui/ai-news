## OpenAI

### Codex モバイルアプリ対応プレビュー（2026-05-14）
ChatGPT モバイルアプリで Codex がプレビュー提供開始。接続された Mac 上で実行中の Codex を承認・確認・方向転換しながら外出先から制御可能に。Enterprise/EDU 向けにはアクセストークン機能も提供し自動化ワークフロー対応を強化。
- ソース: https://releasebot.io/updates/openai

### ChatGPT ファイルライブラリを Free/Go ユーザーへ拡大（2026-05-14）
プラン別ストレージ上限：Free 500MB / Go 4GB / Plus・Business 20GB / Pro 100GB。無料ユーザーでもファイル管理が利用可能になり、入門コストがさらに低下。
- ソース: https://releasebot.io/updates/openai

### ChatGPT 安全性強化：感度の高い会話での文脈認識（2026-05-14）
自殺・自傷関連の会話に「安全性要約」機能を追加。複数会話にわたるリスク信号を認識してより慎重な対応が可能に。Enterprise 向け Codex リモートアクセス（アクセストークン）も同日追加。
- ソース: https://releasebot.io/updates/openai

### Codex Windows サンドボックス実装（2026-05-13）
Windows 環境での Codex 実行を安全化。ファイル書き込みとネットワークアクセスを厳密に制限したサンドボックスを導入し、企業内利用のセキュリティ要件に対応。
- ソース: https://releasebot.io/updates/openai

# AI News Digest 2026-03-29

## OpenAI

### Astral買収 — コーディングツール強化
OpenAIがAstral（Python開発ツール会社）を買収。`uv`・`ruff`を手がけるチームをCodexプロジェクトに統合予定。コーディングエージェント領域への注力が続く。
ソース: https://www.pymnts.com/acquisitions/2026/openai-continues-coding-push-with-astral-acquisition/

### Tool Search API — トークン削減・高速化
新API機能「Tool Search」を導入。従来は全ツール定義をシステムプロンプトに記載していたが、モデルが必要に応じて動的にツール定義を取得する仕組みに変更。多ツール構成のAIアプリでコスト・レイテンシ削減が期待できる。
ソース: https://releasebot.io/updates/openai

### SoftBank $400億ローン — OpenAI IPOへの布石
SoftBankがOpenAIへの$300億出資をカバーするため$400億ローンを調達。OpenAIのバリュエーションは$8,400億、週次アクティブユーザー9億人。2026年中のIPO実現を示唆する動き。
ソース: https://techcrunch.com/2026/03/27/why-softbanks-new-40b-loan-points-to-a-2026-openai-ipo/

ソース: https://www.kdnuggets.com/openclaw-explained-the-free-ai-agent-tool-going-viral-already-in-2026

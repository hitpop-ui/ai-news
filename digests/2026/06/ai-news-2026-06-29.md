# AI News Digest 2026-06-29

## OpenAI ★★★

### GPT-5.6 Sol / Terra / Luna — 限定プレビュー公開（2026-06-26）

GPT-5.6 シリーズを3モデル体制でリリース。初回は約20社の「信頼パートナー」に限定。トランプ政権の大統領令（6/2）に基づきUSG承認後の段階的展開で、一般公開は「数週間以内」予定。

| モデル | 用途 | 価格（入力/出力 per 1M） |
|---|---|---|
| Sol（旗艦） | コーディング・科学推論・長期エージェント | $5 / $30 |
| Terra（バランス） | 日常業務、GPT-5.5 同等性能で2倍安 | $2.50 / $15 |
| Luna（低コスト） | 大量処理・高速・最安 | $1 / $6 |

- **Sol の新推論モード**: `max`（拡張シングルチェーン）・`ultra`（複数サブエージェントへの並列分散）
- API は当面トラステッドパートナーのみ。ChatGPT への展開時期は未発表
- Source: https://openai.com/index/previewing-gpt-5-6-sol/ / https://venturebeat.com/technology/openai-unveils-gpt-5-6-sol-terra-and-luna-models-but-only-accessible-to-limited-preview-partners-for-now-per-us-gov

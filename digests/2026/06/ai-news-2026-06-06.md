# AI News Digest 2026-06-06

## Claude Code

### Claude Code 2.1.163 + 2.1.165（6/4〜5）
2.1.163：managed settings に `requiredMinimumVersion`/`requiredMaximumVersion` 追加（企業が許可バージョン範囲を強制可能）。`/plugin list` コマンド（`--enabled`/`--disabled` フィルター付き）追加。`/btw` に Markdown コピーショートカット（c キー）追加。Stop/SubagentStop フックが `hookSpecificOutput.additionalContext` を返してエラーラベルなしでターンを継続可能に。スキルの `\$` エスケープ構文追加。バックグラウンドセッションがバックグラウンドで自動更新（再接続時のコールドリスタート不要）。`claude -p` が最終結果後にバックグラウンドシェルを ~5秒後に停止するよう修正（ハング修正）。Bedrock/Vertex/Foundry 上で `CI=true` かつ Anthropic API キーなし時に `claude -p` が失敗するバグ修正ほか多数。
2.1.165（6/5）：バグ修正・安定性向上のみ。
- Source: https://code.claude.com/docs/en/changelog

# AI News Digest 2026-06-20

## Claude Code

### Claude Code 2.1.183（2026-06-19）— 破壊的操作の自動ブロック
- 明示的指示なしに `git reset --hard`・`git checkout -- .`・`git clean -fd`・`git stash drop` をブロック
- `terraform destroy` / `pulumi destroy` / `cdk destroy` も同様に制限（特定スタックを明示した場合のみ実行可）
- 廃止予定モデルへのリクエスト時に警告を表示
- `attribution.sessionUrl` 設定追加：コミット・PRからセッションURLリンクを省略可能
- `/config --help` で利用可能なショートハンドキー一覧を表示
- サブエージェント内で WebSearch が空の結果を返すバグを修正
- tmux teammate ペーン起動失敗・Remote Control セッション安定性も改善

ソース: https://code.claude.com/docs/en/changelog

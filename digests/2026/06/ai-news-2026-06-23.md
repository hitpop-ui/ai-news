## 主要トピック

### Claude Code 2.1.186 リリース（2026-06-22）
前バージョン（2.1.185）から新機能追加。

**MCPサーバー認証の簡素化**
- `claude mcp login <name>` / `claude mcp logout <name>` コマンドを追加
- インタラクティブな `/mcp` メニューを開かずにCLIから直接認証可能
- `--no-browser` フラグ付きでSSH経由実行にも対応
- `claude mcp get/remove` でタイプミス時に近いサーバー名を提案

**Bashコマンド自動応答（重要な挙動変更）**
- `!` で実行するbashコマンドの出力にClaudeが自動応答するよう仕様変更
- 従来動作に戻す場合は `settings.json` で `"respondToBashCommands": false` を設定

**ワークフロー・プラグイン改善**
- `/workflows` エージェント詳細ビューにステータスフィルタリング（`f` キー）追加
- `/plugin` インストール済みタブに「スキル」セクション追加

**設定追加**
- `teammates: "iterm2"` — macOS iTerm2統合
- `awsAuthRefresh` — AWS認証リフレッシュオプション

**主なバグ修正**
- マシン起動時の「Content block not found」エラー修正
- サブエージェントのトランスクリプトスクロールがメイン画面に影響する問題修正
- `Agent(type)` deny ルールが名前付きサブエージェント生成時にも適用されるよう修正
- メモリ索引がサイズ制限に近づいた際のコンパクション推奨メッセージ追加

ソース: https://code.claude.com/docs/en/changelog

## Claude Code

### v2.1.187 (2026-06-23) + v2.1.190 (2026-06-24)

**v2.1.187** — セキュリティ強化・大規模バグ修正パッチ

- **`sandbox.credentials` 設定追加**: サンドボックス内コマンドが認証情報ファイル・秘密環境変数を読み取れないようブロック。チームや副業での認証情報漏洩リスクを軽減
- **org設定によるモデル制限**: モデルピッカー・`--model`・`/model`・`ANTHROPIC_MODEL` の全4経路に組織ポリシーが適用可能に
- フルスクリーンモードでセレクトメニューのマウスクリック対応
- バグ修正多数: `--resume` エラー・`--json-schema` 無限ループ・リモートMCPタイムアウト(5分)・CJK文字化け・`/update` ハング・バックグラウンドジョブ無限"working"スタック・チャンネル接続ドロップ など
- `/btw` 改善: ←/→矢印キーで前の回答をステップスルー可能に
- `/plugin` 改善: 最近使っていないプラグインを表示
- `/install-github-app` 改善: GitHub Actionsワークフロー設定がオプション化

**v2.1.190** — バグ修正・信頼性向上（詳細は別途公開）

ソース: https://code.claude.com/docs/en/changelog

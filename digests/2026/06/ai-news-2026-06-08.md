# AI News Digest 2026-06-08

## Claude Code

### Claude Code 2.1.166（2026-06-06）— セキュリティ強化 + フォールバックモデル

前回確認（2.1.165）からの主な変更点：

- **`fallbackModel` 設定追加**：最大3つのフォールバックモデルを順序で設定可能。プライマリモデルが過負荷・利用不可時に自動切替。`--fallback-model` フラグはインタラクティブセッションにも対応
- **denyルールにグロブパターン追加**：ツール名位置に `"*"` を指定することで全ツールを一括拒否可能に
- **クロスセッションメッセージのセキュリティ強化**：他の Claude セッションから `SendMessage` 経由で中継されたメッセージはユーザー権限を持たない。受信側は中継された権限リクエストを拒否し、Auto モードはブロック
- **Thinking 無効化オプション**：`MAX_THINKING_TOKENS=0`・`--thinking disabled`・モデル別設定で、デフォルトでthinkingするモデルのthinkingを無効化
- `claude update` がダウンロード前にターゲットバージョンを表示
- `claude agents` のURLフィルタリング機能追加
- バグ修正：画像レンダリングエラー・追加トークン使用の修正、リモートセッションのロック修正、JetBrains IDEターミナルのちらつき修正（IntelliJ / PyCharm / WebStorm 2026.1+）、Windows PowerShellコマンド検証のハング修正

2.1.167・2.1.168（同日）はバグ修正・安定性向上のみ。

- Source: https://code.claude.com/docs/en/changelog

# AI ニュースダイジェスト 2026-06-11

## Claude / Anthropic

### Claude Code 2.1.169：セーフモード・/cd コマンド追加（昨日分補足）
- `--safe-mode` フラグ（環境変数 `CLAUDE_CODE_SAFE_MODE`）: 全カスタマイズを無効にしてトラブルシューティング起動が可能に
- `/cd` コマンド: セッション中にプロンプトキャッシュを壊さず作業ディレクトリを変更できる
- `disableBundledSkills` 設定: バンドルスキルの非表示オプション追加
- リモートセッション・バックグラウンドエージェント改善、エンタープライズポリシー適用強化、CPU使用量削減
- https://releasebot.io/updates/anthropic/claude-code

### Claude Managed Agents パブリックベータ（6/9）
- スケジュール実行（cron ベース）と環境変数ボルト（Vault）機能がパブリックベータ公開
- エージェントが API キーをモデルに渡さず、Vault 経由で CLI ツールに安全アクセス可能
- 「毎朝〇〇する」型の定期自動化がインフラ不要で構築できるようになった
- https://releasebot.io/updates/anthropic

## Cursor

### Bugbot 大幅高速化（6/10）
- 平均レビュー時間: 約 5 分 → 約 90 秒（Composer 2.5 モデル導入で実現）
- バグ検出率: 0.56 → 0.62 件/レビューに向上
- コスト: 約 22% 削減
- 新機能: `/review` コマンドでプッシュ前に Bugbot + Security Review を一括実行可能
- GitHub/GitLab 統合で前回レビュー済み diff を記憶し、新規変更のみレビュー可能
- https://cursor.com/changelog

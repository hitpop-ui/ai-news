# AI News Digest 2026-06-09

## Apple WWDC 2026 基調講演（6/8）— AI開発者向け重要発表

### Siri 2.0：Google Gemini 120兆パラメータ採用

Apple が Google と協業し Siri を全面刷新。クラウド版は Gemini 大規模モデル、オンデバイス版は蒸留モデルをニューラルエンジンで実行。マルチターン会話・ウェブ検索・画面認識・ファイル解析に対応。プライベートクラウドコンピュートで暗号化・隔離。

- Source: https://chatforest.com/builders-log/wwdc-2026-keynote-confirmed-apple-ai-platform-builder-guide/

### Apple Intelligence Extensions：Claude / ChatGPT がデフォルト AI に選択可能

iOS 27 / macOS 27 の設定で、Gemini（デフォルト）・**Claude**・ChatGPT からメイン AI プロバイダーを選択可能になった（1デバイス1プロバイダーの"binary"設計）。Writing Tools・Image Playground 等を選択プロバイダーが処理する。

- Source: https://chatforest.com/builders-log/wwdc-2026-keynote-confirmed-apple-ai-platform-builder-guide/

### Core AI：Core ML を置き換える新フレームワーク

LLM・生成 AI 向けに Core ML を刷新した **Core AI** フレームワーク発表。ストリーミング生成・会話履歴保持・動的ルーティングをネイティブサポート。既存 Core ML コードはそのまま動作するが、新規開発は Core AI 推奨。

- Source: https://chatforest.com/builders-log/wwdc-2026-keynote-confirmed-apple-ai-platform-builder-guide/

### MCP プラットフォーム全体展開（iOS 27 / macOS 27）

Xcode 限定だった MCP（Model Context Protocol）対応が、iOS 27 / macOS 27 で OS レベルに昇格。**Siri 2.0 と Core AI レイヤーが登録済み MCP サーバーを直接呼び出せる**ようになる。外部エージェント（Claude Agent 含む）と Apple プラットフォームの統合が大幅に深化。

- Source: https://chatforest.com/builders-log/wwdc-2026-keynote-confirmed-apple-ai-platform-builder-guide/

### Foundation Models フレームワーク拡張

オンデバイス LLM API が拡張。より大規模なモデル・コンテキスト拡大・ファインチューニングに対応。API は引き続き**無料**。

- Source: https://chatforest.com/builders-log/wwdc-2026-keynote-confirmed-apple-ai-platform-builder-guide/

### Xcode 26.3（2月既出）：MCP経由で Claude Agent / Codex と連携

2月リリース済みの Xcode 26.3 が今回改めてスポットライト。`mcpbridge` バイナリ経由で20のXcodeツールを外部エージェントに公開。**Claude Agent と OpenAI Codex** がネイティブ対応済み。ファイル作成・ビルド・テスト実行・Apple開発者ドキュメント参照が可能。

- Source: https://byteiota.com/wwdc-2026-developer-preview-what-ships-june-8/

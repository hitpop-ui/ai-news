## 🔥 最重要：Anthropic、6月15日にプラン利用枠を分割へ

### Claude 有料プランの「対話」と「自動化」が完全分離（2026-06-15 実施）

Pro / Max / Team 全プランで、月次利用枠の使い方が大幅変更される。

| プラン | 月額料金 | Agent SDKクレジット |
|--------|---------|-------------------|
| Pro | $20 | $20/月 |
| Max 5x | $100 | $100/月 |
| Max 20x | $200 | $200/月 |
| Team Standard | $25/シート | $20/月 |
| Team Premium | $125/シート | $100/月 |

**クレジット消費対象（新枠）**
- Claude Agent SDK（Python / TypeScript）
- `claude -p`（非対話モード）
- Claude Code GitHub Actions
- Agent SDK 経由のサードパーティ（OpenClaw 等）

**影響なし（従来枠）**: Web チャット・モバイルアプリ・対話型 Claude Code

**運用ルール**
- 個人単位付与・チーム間共有不可
- 月末リセット（繰り越しなし）
- **初回はオプトイン必須**（手動で有効化）
- 超過後は従量課金設定に依存

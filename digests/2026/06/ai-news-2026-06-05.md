# AI News Digest 2026-06-05

## Claude Code

### Claude Code 2.1.162（2026-06-03）
- `claude agents --json` に `waitingFor` フィールド追加。ブロック中セッションが「何を待っているか」（権限プロンプト等）をAPI経由で確認可能に
- **Windsurf が「Devin Desktop」にリブランド**。`/ide`・`/terminal-setup`・`/scroll-speed` メニューの表記が変更
- スラッシュコマンドのオートコンプリートをクリックすると入力欄に挿入→Enterで実行に変更（以前は即実行）
- Remote Control がスタートアップメッセージからセッションリンク付きの永続フッターピルに変更
- バグ修正: WebFetch権限ルール修正、Windowsパスマッチング問題、MCP per-serverタイムアウト設定不具合など
- https://code.claude.com/docs/en/changelog

## Cursor

### Cursor 3.7 — Canvas Design Mode ＋ Context Usage Report（2026-06-04）
- **Design Mode**: キャンバス内でUI要素を直接選択・注釈付け可能。テキスト説明不要で視覚的に「ここを変えて」と指示できるように
- **Context Usage Report**: システムプロンプト・ツール定義・ルール・スキル等のトークン配分をブレークダウン表示。コスト最適化に有効
- 共有キャンバスのフルスクリーン表示、キャンバスへのボタン埋め込み（クリックで特定プロンプト実行）
- https://cursor.com/changelog

### Cursor Enterprise Organizations GA（2026-06-03）

- 部署・地域ごとの運用ユニット設定、複数チーム横断のユーザー管理、組織レベルのIDP管理と利用分析
- https://cursor.com/changelog

## Devin

### 個人オートメーション ＋ Devin Review 強化（2026-06-03）
- **個人オートメーション**: 自分のアイデンティティで実行される自動化タスク。専用トグル・権限モデル・バッジ付き
- **Devin Review**: GitHub Enterprise Server・GitLab PRでも「Devin Reviewで開く」に対応。PR単位の支出制限を再設計（ソフトブロック）
- テスト前承認: 設定で「常に承認」にすれば毎回のプロンプト不要
- **V3 API**: 組織メンバー一覧エンドポイント（`GET /v3beta1/organizations/{org_id}/members`）追加、オートメーションCRUD API完備
- SSO/SCIM: JITプロビジョニング切り替え可能、SSOのみのエンタープライズユーザーが自動リダイレクト対応
- https://docs.devin.ai/release-notes/overview

# AI News Digest 2026-06-17

## Claude Code

### Claude Code 2.1.178 リリース（2026-06-15）
前回記録の 2.1.176 以降でリリースされた最新版。主な変更点：
- **Permission Rules 拡張**：`Tool(param:value)` 構文でツール入力パラメータ単位のマッチングが可能に（ワイルドカード `*` 対応）。例：`Agent(model:opus)` でOpusサブエージェントのみをブロック
- **ネストされた `.claude/` サポート**：スキルをネストされたディレクトリから読み込み可能。名前競合時は `<dir>:<name>` で両方利用可
- **Auto mode 強化**：サブエージェント起動前に分類器で評価し、ブロック対象アクションを事前防止
- `/doctor` UI改善（フラットツリー・アイコン明確化）、`/rc failed` インジケーターでRemote Controlエラーを常時表示

ソース: https://code.claude.com/docs/en/changelog

## Devin

### Windsurf → Devin Desktop にリブランド（2026-06-02・追跡開始）
前回の確認状態に未収録だった重要イベント。WindsurfがOTAアップデートで自動的にDevin Desktopへ移行：
- 既存の設定・拡張・キーバインドはそのまま引き継ぎ（再インストール不要）
- 起動画面が **Agent Command Center**（全エージェントのカンバン）に変更
- **Devin Local**（Cascadeの後継）：30%トークン効率向上・Rust書き直し・サブエージェント対応
- **ACP（Agent Client Protocol）** をオープンソースで採用——ACP対応エディタならどのエージェントでも実行可
- Cascade は 2026-07-01 廃止予定

ソース: https://devin.ai/blog/windsurf-is-now-devin-desktop/

### Devin v3 API 正式 GA・Desktop テスト機能追加（2026-06-12〜）
- **v3 API** がベータ卒業：RBAC・セッション帰属・新機能をフルサポート
- **Computer Use for Desktop**：Linux上で起動するデスクトップアプリのE2Eテストを自動化。Devinがアプリを操作し編集済み録画を共有

ソース: https://docs.devin.ai/release-notes/2026

OpenClawベースの常時稼働エージェント「Scout」（Autopilotカテゴリ第一弾）のWork IQ APIが本日一般提供開始：

## 上位コンテンツの傾向まとめ

### 1. 「Claude Code 使い方」検索上位の傾向

**支配的なコンテンツ形式：初心者向け完全ガイド系**

- 「インストールから実践まで」「始め方・料金・できること」をセットで解説する記事が上位を独占
- 非エンジニア向け・副業向けにフォーカスした記事が増加（「Claude Codeで稼ぐ完全ガイド 月30万円」等）
- note での個人発信も増加傾向（「最初の1万円を稼ぐ5ステップ」等）

**勝ちパターンのキーワード組み合わせ：**
「初心者 / 非エンジニア」×「2026年最新」×「副業 / 稼ぐ / 料金」

**空白地帯：**
- 「週5会社員が平日夜だけでClaude Codeを使う具体的な使い方」という切り口はまだ少ない
- チーム・組織導入向けではなく「一人副業」向けの具体的ワークフロー記事が不足

---

### 2. 「AI副業」検索上位の傾向

**共通の訴求ポイント：月5万円**

- ほぼすべての上位記事が「初心者が月5万円を目指す」というゴール設定を採用
- 副業ジャンルとして多く登場するもの：ライティング・SNS運用代行・Web制作・プロンプト販売
- 「AIによる効率化」と「人間の文脈理解」の掛け合わせが差別化ポイントとして言及される

**コモディティ化の示唆：**
- 「2026年はAI副業がコモディティ化直前」という表現が複数記事に登場
- 早期参入者が有利ポジションを確保できるという論調

**空白地帯：**
- 会社員が本業リスクなく副業を始める「情報発信型副業」への具体的な導線が少ない
- Claude Code × 具体的副業ジャンルの掛け合わせ記事がまだ少ない

---

### 3. 「ChatGPT活用 副業 ブログ SNS」検索上位の傾向

**ブログアフィリエイトとSNS運用代行が2大ジャンル**

- ChatGPT × ブログアフィリエイトで月20万円達成事例あり（ゆけむりブログ等）
- SNS運用代行は「投稿文生成 + スケジューリング自動化」がセット提案
- 「AIが書いたまま納品してはいけない」という品質注意喚起が増加

**トレンド変化：**
- 2025年まで：ChatGPTで記事を書く → 2026年：AIエージェントでブログ全体を管理する方向へシフト
- SNS運用代行もテキスト生成から「アカウント戦略設計 + AI実行」の高付加価値化が進む
